# MVP Scope

The MVP of Smart Inquiry AI focuses on one complete inquiry workflow.

## MVP Goal

Help a business receive a customer inquiry, understand it quickly with AI assistance, and track basic follow-up status.

## Included In MVP

- Public inquiry form
- Customer contact details
- Request or project description
- Inquiry record storage
- AI output using the initial MVP contract
- Email notification for new inquiry
- Internal dashboard
- Inquiry detail view
- Basic status tracking

AI generation failure must never prevent an Inquiry from being created.

## MVP Inquiry Lifecycle

- New
- In Progress
- Completed
- Archived

## Suggested Inquiry Fields

- Name
- Email
- Phone
- Company
- Inquiry type
- Budget range
- Timeline
- Message
- Preferred contact method

## Initial AI Output Contract

- summary
- customer_need
- urgency
- missing_information
- suggested_next_action

## MVP Non-Goals

- Full CRM
- Attachment upload
- Payment processing
- Authentication for the first demo implementation
- User role management
- Multi-tenant SaaS architecture
- Advanced workflow automation
- Calendar integration
- Chat integration
- Analytics dashboards
- Custom field builder

## MVP Completion Criteria

The MVP is complete when a team can:

- Submit a new inquiry
- Receive a notification
- View the inquiry in a dashboard
- Read the AI-generated inquiry context when available
- Update the inquiry status
- Decide the next follow-up action
